\# TruthLens - Universal Document Fraud Detection



\## Project Overview

AI-powered system to detect manipulated documents using:

\- Computer Vision (visual tampering)

\- Generative AI (context understanding)

\- Domain-specific validation (business logic)



\## Document Types Supported

\- Financial: Bank statements, invoices, payslips, tax returns

\- Educational: Degrees, certificates, mark sheets

\- Employment: Offer letters, experience letters

\- Identity: Aadhaar, PAN, passport

\- Legal: Contracts, affidavits

\- Medical: Prescriptions, bills



\## Tech Stack

\- Python 3.10

\- OpenCV (Computer Vision)

\- Vision-Language Models (Gen AI)

\- FastAPI/Streamlit (Deployment)
